VAULT PASS

Vault Pass adalah sistem subscription DranBrawl, dengan mengaktifkan/berlangganan subscription maka anda akan mendapat benefit:

BASIC VAULT

    Mulai dari 10K/bulan kamu akan mendapatkan:
  • Akses fitur PRO di DranBrawl
  • Tidak ada batasan Custom Command
  • Cooldown dipercepat
  • Bonus pendapatan
  • Badge khusus di Profile
    • Unlock access DranBrawl Elite di Server anda Bonus Crypto harian Role khusus di Official Server


ROYALE VAULT

    Mulai dari 35K/bulan kamu akan mendapatkan:
  • Akses fitur PRO di DranBrawl
  • Tidak ada batasan Custom Command
  • Cooldown dipercepat
  • Bonus pendapatan
  • Badge khusus di Profile
  • Unlock access DranBrawl Elite di Server anda
  • Bonus Crypto harian
  • Role khusus di Official Server
    • Menguasai Asia Tenggara
